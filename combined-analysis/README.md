# Combined ML Analysis

Applies and compares classification and regression techniques on the same loan dataset to identify the best performing model.

## Tech Stack
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn

## ML Pipeline
1. Data loading and exploration
2. Preprocessing — Label Encoding, Standard Scaling
3. Train/test split
4. Multi-model training and comparison
5. Evaluation — confusion matrix, classification report, accuracy score

## Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Evaluation Metrics
- Confusion Matrix
- Classification Report
- Accuracy Score
