# AIML Internship Projects

Three machine learning projects built during an AIML internship (June 2026) 
using Python and Scikit-learn on a real-world loan dataset.

## Projects

### 1. Classification
Predicts loan approval using Logistic Regression, Decision Tree, 
and Random Forest classifiers.

### 2. Regression  
Predicts loan amount values using Linear Regression, Decision Tree, 
and Random Forest regressors. Best R² Score: 0.60

### 3. Combined Analysis
Applies and compares multiple ML techniques on the same dataset, 
evaluated using confusion matrix, classification report, and accuracy score.

## Tech Stack
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn

## Tools
Google Colab, GitHub
